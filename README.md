# Estudos e Exercícios

Este repositório contém repositórios desenvolvidos durante cursos e estudos pessoais.  

Cada repositório representa a área de estudo. Com dentro delas possuindo os diferentes cursos relacionados as mesmas.   

Todos os repositórios foram feitos seguindo as aulas, porém sempre com algumas alterações que eu achasse que ficaria melhor para meu aprendizado e até melhor entendimento e lógica.  

Os códigos estão comentados para que sempre que seja necesário eu consultar o que foi aprendido, ele estará lá.
